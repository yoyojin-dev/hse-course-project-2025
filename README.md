## 🎮 Как играть

1. Откройте главную страницу и создайте игру.
2. Передайте сгенерированную ссылку вида `/joining/{code}` другим игрокам.
3. Каждый игрок вводит свой никнейм и попадает на страницу игры.
4. После подключения минимум двух игроков нажмите **«Старт игры»**.
5. Игроки делают ходы по очереди, нажимая кнопку **«Сделать ход (монетка)»**.
6. Побеждает тот, кто первым наберет 5 очков!

## 🚀 Разработка и запуск локально

Проект состоит из бэкенда (Go) и фронтенда. Для удобного запуска используется Docker.

### Запуск проекта
```shell
make up
# или
docker compose up --build
```

### Остановка
```shell
make down
# или
docker compose down
```

### Просмотр логов
```shell
make logs
# или
docker compose logs -f
```

## 📡 API (Основные маршруты)

- `GET /api/hello` - Проверка доступности API
- `POST /api/create` - Создать новую игру
- `POST /api/join` - Войти в игру по коду и никнейму
- `GET /api/game/{code}` - Получить текущее состояние игры
- `POST /api/game/{code}/start` - Начать игру
- `POST /api/game/{code}/move` - Сделать ход

## 🌍 Деплой на VPS

Проект поддерживает автоматический деплой через GitHub Actions (см. `deploy/README.md`).
Для ручного деплоя на сервере выполните:
```shell
cd /opt/featureban/app
bash deploy/deploy.sh
```
